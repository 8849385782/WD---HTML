What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create the structure of web pages. It is not a programming language — it doesn't compute logic — it describes content and how that content is organized, using a system of elements made of tags.

Basic Syntax

Most HTML elements have an opening tag, some content, and a closing tag:

<p>This is a paragraph.</p>

Some elements are "self-closing" or "void" — they don't wrap content, e.g. <hr> and <br>.

Attributes give extra information inside the opening tag: <img src="cat.jpg" alt="A cat">.

Document Structure

Every HTML5 page follows the same skeleton:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <!-- Visible page content goes here -->
  </body>
</html>
  • <!DOCTYPE html> — tells the browser this is an HTML5 document.
  • <html> — the root element wrapping everything.
  • <head> — metadata: title, character set, linked CSS, etc. Nothing here is visible on the page.
  • <body> — everything the visitor actually sees.

Basic Tags Covered

  • <h1> through <h6> — headings, in decreasing importance/size.
  • <p> — a paragraph of text.
  • <hr> — a horizontal rule, used to visually separate sections.
  • <!-- comment --> — a comment; ignored by the browser, visible only in the source code.
