Embedding Video, Audio & YouTube

<video>

Embeds a self-hosted video file directly on the page.

<video src="clip.mp4" controls width="640"></video>

<audio>

Embeds a sound file with a built-in player.

<audio src="song.mp3" controls></audio>

YouTube <iframe>

YouTube videos aren't embedded with <video> — instead you copy YouTube's own embed code, which is an <iframe> pointing at youtube.com/embed/VIDEO_ID. Get it from a video's Share → Embed button.

<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube video player" frameborder="0"
  allowfullscreen></iframe>

Key Attributes

  • controls — shows the browser's built-in play/pause/volume/seek bar. Without it, the media has no visible UI at all.
  • autoplay — starts playback as soon as the page loads. Most browsers block autoplay with sound unless the element is also muted, so autoplaying video almost always needs muted alongside it.
  • loop — restarts playback automatically from the beginning once it ends.
  • controlsList — (video only) hides specific parts of the default control bar, e.g. controlsList="nodownload nofullscreen" removes the download and fullscreen buttons while keeping play/pause/volume.
